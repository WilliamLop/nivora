# Kit operativo para setters

## Objetivo

Entrenar setters principiantes para convertir leads frios en reuniones agendadas para landing pages y sitios web.

La regla del sistema es simple:

- el setter no cierra ventas
- el setter no cotiza proyectos complejos en llamada fria
- el setter si detecta dolor, valida interes y busca dejar una reunion agendada

## Oferta de entrada

Para esta fase trabajamos solo con dos productos:

- `landing page` como oferta principal de entrada
- `sitio web` como alternativa cuando el negocio necesita una presencia mas completa

Eso evita dispersar al equipo con demasiados servicios al mismo tiempo.

## Como usar este kit

- dia 1: estudiar producto y diferencia entre landing y sitio web
- dia 2: practicar apertura, diagnostico, lenguaje comercial y objeciones
- dia 3: hacer roleplays y llamadas supervisadas

Durante las primeras dos semanas:

- revisar varias llamadas por persona
- corregir tono, claridad, preguntas y cierre a reunion
- medir reuniones agendadas, reuniones asistidas y ventas cerradas

## Regla de oro del setter

El setter no necesita sonar tecnico ni "vender paginas".

Necesita sonar claro y util:

- detectar si el negocio esta perdiendo oportunidades
- conectar ese dolor con una solucion simple
- mover la conversacion al siguiente paso correcto

Si hay interes real, el trabajo esta bien hecho cuando la reunion queda agendada y el closer recibe un buen brief.
