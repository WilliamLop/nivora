# Anexo 3 - Handoff y metricas

## Objetivo

Hacer que cada reunion agendada llegue al closer con contexto suficiente para vender bien.

Una reunion sin brief claro obliga al closer a empezar de cero.
Eso baja conversion y rompe la confianza en el setter.

## Brief obligatorio por reunion agendada

Usa este formato minimo:

```text
Setter:
Fecha de llamada:
Negocio:
Contacto:
Ciudad / nicho:
Telefono / WhatsApp:
Correo:
Presencia actual:
Oferta mas probable: landing / sitio web
Dolor principal detectado:
Servicio o prioridad que quieren mover:
Objecion principal:
Nivel de interes: bajo / medio / alto
Quien decide:
Fecha y hora de reunion:
Canal de reunion:
Notas extra:
```

## Checklist de handoff de calidad

Antes de dar por buena una reunion, confirma:

- el nombre del contacto esta claro
- el canal de reunion esta confirmado
- se entiende por que acepto la reunion
- hay al menos un dolor o necesidad concreta
- esta marcada la oferta mas probable
- las objeciones ya estan registradas

## Regla operativa

Si no hay brief completo, la reunion no cuenta como handoff de calidad.

Eso no significa cancelar la reunion.
Significa que hay que corregir el proceso del setter.

## Definiciones de metricas

| Metrica | Definicion |
| --- | --- |
| `llamadas realizadas` | total de intentos hechos por el setter |
| `contactos efectivos` | conversaciones reales de mas de 30 segundos |
| `reuniones agendadas` | reuniones con fecha, hora y canal confirmados |
| `show rate` | reuniones asistidas / reuniones agendadas |
| `ventas cerradas por setter` | ventas cerradas que vienen de reuniones agendadas por ese setter |
| `primer pago cobrado` | monto inicial efectivamente cobrado del proyecto |

## Lectura correcta del tablero

- muchas llamadas y pocas conversaciones: problema de data, horarios o apertura
- muchas conversaciones y pocas reuniones: problema de diagnostico, valor o cierre
- muchas reuniones y poco show rate: problema de confirmacion o seguimiento
- muchas reuniones asistidas y pocas ventas: problema de fit, handoff o cierre

## Cadencia recomendada de revision

### Diario

- revisar llamadas clave
- revisar reuniones del dia siguiente
- corregir notas flojas

### Semanal

- comparar setters
- detectar objeciones repetidas
- medir show rate y cierres
- ajustar guion si hace falta

## Semaforo de calidad

Usa este criterio simple:

- verde: agenda reuniones con brief completo y show rate sano
- amarillo: agenda, pero con notas flojas o mala confirmacion
- rojo: habla mucho, agenda poco o entrega reuniones sin contexto

El objetivo no es castigar.
Es saber donde entrenar mejor.
